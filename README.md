# Meninas.comp

Official website for the **Meninas.comp – Computação também é coisa de menina** extension project.

---

## 🔨 Features

- **Home Page**: presentation of the project, mission, and impact.  
- **Mascot Section (Ame Bit Byte)**: storytelling about the project inspiration.  
- **Team Section**: presentation of professors and contributors with links (e.g., Lattes).  
- **Testimonials Carousel**: dynamic section showing participants' experiences.  
- **Partners & Sponsors**: logos and institutional support.  
- **Responsive Design**: optimized for mobile, tablet, and desktop.  
- **Interactive Elements**: animations and hover effects for better user experience.  

---

## ✔️ Technologies Used

- `HTML5` → semantic structure  
- `TailwindCSS` → responsive and modern styling  
- `JavaScript` → interactivity (carousel, navigation, etc.)  
- `Flexbox & Grid` → layout organization  
- `Responsive Design` → multi-device support  
- `Accessibility` → readable and inclusive interface
